

## 机器人内置函数介绍

### 基础动作类
- **前进动作**
- **后退动作**
- **左转圈动作**
- **右转圈动作**
- **左平移动作**
- **右平移动作**
- **点头动作**
- **摇头动作**
- **控制小车灯颜色**
- **关闭车灯的动作**
- **设置任务失败标志**
	- 说明:可以提前预知失败无法完成的任务，直接使用
- **播放音乐的接口**
- **获取面前物体/障碍物的距离**
	 - 说明：只能测量前方的障碍物
- **根据对应的距离进行相关动作**
	- 说明:根据距离进行避障、停止、跟随


### 视觉追踪类
- **描述周围的场景**
- **追踪人脸或跟随人脸**  
  需要注意，只有指令明确说才去调用此指令。
- **颜色巡线**
 一共有红:red,绿:green,蓝:blue,黄:yellow,黑:black ,只有出现带颜色相关的指令明确说才去调用此指令。
- **颜色跟随的接口动作**:
有红:red,绿:green,蓝:blue,黄:yellow，只有出现带颜色相关的指令明确说才去调用此指令。
- **追踪物体的接口动作**:
只有指令明确说才去调用此指令，此接口也能追踪除了人脸外其它的人体部位。
